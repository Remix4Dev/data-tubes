Column

Description
Description | Methods (details) Constants (details)

A Column stores the definition of one column of the DataTable.

For each column that will be used in the data of the DataTable, a column-definiton has to be added to the DataTable.

Please visit Google's API documentation at https://google-developers.appspot.com/chart/interactive/docs/reference#dataparam for detailed background information.
