HAxis

Extends \googlecharttools\view\options\Axis

The HAxis defines the appearance of the chart's horizontal axis.

The options set in this option-set are only valid in discrete charts.

boolean

Arguments

$angle

The angle used to display the label in deg. If set to null,

This allows to set additional options that are currently not supported by the set-methods. However, you should always use on of the other set-methods if possible.
