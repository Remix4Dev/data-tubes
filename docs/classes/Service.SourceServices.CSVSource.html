Module to filter and sort data before returning to requester.

applyParameters(mixed[] $aarray) : mixed[]
inherited

Should be applied to all inherited classes to ensure consistent support regarding parameter filters. It only filters data; it does NOT transform data, e.g., filling in dates, cumulative totals, etc. That should be done in the implementer classes. It does NOT apply filters to all or single data items (as it doesn't make sense to do so).

$aarray

mixed[]

The reshaped array that has only the needed data for the request.

Response

mixed[]

An associative array of data, with null values removed and filters and other parameters applied, sorted by field of interest.
