ChartArea

Extends \googlecharttools\view\options\OptionStorage

The ChartArea defines the size and position of the displayed chart area.

view
subpackage

__construct(mixed $left = null, mixed $top = null, mixed $width = null, mixed $height = null)

An integer or a string that contains a number followed by a percent sign can be assigned to each argument. An integer or a number-only string is interpreted as pixel value, a number followed by a percent sign as percentage value.

setPosition(mixed $left = null, mixed $top = null)

An integer or a string that contains a number followed by a percent sign can be assigned to the arguments. An integer or a number-only string is interpreted as pixel value, a number followed by a percent sign as percentage value.

setOption(string $name, string $value)
inherited

This allows to set additional options that are currently not supported by the set-methods. However, you should always use on of the other set-methods if possible.

Through this method, additional option-code can be added to the generated JavaScript-code. This is usefull, when options should be set that are currently not directly supported by this API (through set...() methods).

Arguments

$name

string

The option's name.

$value

string

The option's value. If set to null, the option

         will be removed.
