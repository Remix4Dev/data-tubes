Row

Description
Description | Methods (details)

A Row represents a data entry (a row) of the DataTable.

Each row contains one or more Cells that stores the actual data of this row. The cells of each row have to match the Column definition. Column and Cell are connected by the order they are added to the DataTable and Row, respectively. That is, the Cell added first belongs to the Column definition that has been added first, the second-added Cell to the second-added Column, and so on. Therefore, no more cells per row can be added then columns have been defined.
