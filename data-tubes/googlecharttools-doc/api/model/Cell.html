Cell

Description
Description | Methods (details)

A Cell stores the data for one cell in the DataTable.

Various cells are grouped together in a Row.

Please visit Google's API documentation at https://google-developers.appspot.com/chart/interactive/docs/reference#cell_object for detailed background information.
