DataTable

Description
Description | Methods (details)

A DataTable represents the data that will be visualized in a chart.

Google Chart Tools uses a table-format as data-storage. The actual function of the table's head and cells depends on the chart the data is used for (e. g. in pie-charts, a table with two rows is used where the first column represents the slices' label. The second column is used as the slices' value).

The DataTable is divided into a column-definition (the table's header) and rows. By the column definition, the column's value-type (string, number, etc.) is specified and the data (that will be visualized) is stored in the rows.

Example: Lets assume the table below should be stored in a DataTable.

 |----------------|-----------------------|
 | Name: Task     | Name: Hours per Day   |
 | Type: string   | Type: Number          |
 |----------------|-----------------------|
 |    Work        |   11                  |
 |    Eat         |   2                   |
 |    Commute     |   2                   |
 |    Watch TV    |   2                   |
 |    Sleep       |   7                   |
 |----------------|-----------------------|

This table will be represented by the DataTable through the following code:

  1.  $labelColumn = new Column(Column::STRING, "t", "Task");
  2.  $valueColumn = new Column(Column::NUMBER, "h", "Hours per Day");
  3.  
  4.  $data = new DataTable();
  5.  $data->addColumn($labelColumn);
  6.  $data->addColumn($valueColumn);
  7.  
  8.  $workRow = new Row();
  9.  $workRow->addCell(new Cell("Work"));
  10.  $workRow->addCell(new Cell(11));
  11.  $data->addRow($workRow);
  12.  
  13.  $eatRow = new Row();
  14.  $eatRow->addCell(new Cell("Eat"));
  15.  $eatRow->addCell(new Cell(2));
  16.  $data->addRow($eatRow);
  17.  
  18.  $commuteRow = new Row();
  19.  $commuteRow->addCell(new Cell("Commute"));
  20.  $commuteRow->addCell(new Cell(2));
  21.  
  22.  ...

You will find detailed information on how many columns are supported by a specific chart, their purpose etc. in detail at the corresponding chart's subpage published at https://google-developers.appspot.com/chart/interactive/docs/gallery.

Fills the DataTable by reading an associative array.


Every element of the array will be seen as a new row whereby the  array's keys are used for the first column and the array's values  for the second column. Thus, this will require a DataTable with two columns  only.
