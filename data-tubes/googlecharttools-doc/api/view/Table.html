Table

Description
Description | Methods (details) Constants (details)

Creates a table.

Data format:
The DataTable is directly mapped to the table. The DataTable's Columns are used for the table's header.

See https://google-developers.appspot.com/chart/interactive/docs/gallery/table for examples and detailed background information on the required data format.
