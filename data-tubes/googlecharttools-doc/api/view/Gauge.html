Gauge

Description
Description | Methods (details)

Creates a gauge.

Data format:
The gauge requires a DataTable which has two columns. The first column is used for the gauge's label and and the second one for its value. Each Row will be used for one gauge.

See https://google-developers.appspot.com/chart/interactive/docs/gallery/gauge for examples and detailed background information on the required data format.
