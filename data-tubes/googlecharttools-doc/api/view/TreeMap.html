TreeMap

Description
Description | Methods (details)

Creates a tree map.

Data format:
The tree map requires a DataTable with at least three columns. The first column is used as the nodes ID and name. The second column sets the nodes parent ID. The nodes' size is defined through the third column. The fourth column is optional and is used to calculate the nodes' color.

See https://google-developers.appspot.com/chart/interactive/docs/gallery/treemap for examples and detailed background information on the required data format.
