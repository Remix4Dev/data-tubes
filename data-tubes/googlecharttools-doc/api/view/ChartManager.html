ChartManager

Description
Description | Methods (details)

The ChartManager manages all charts that will be displayed on the page.

The ChartManager is responsible for the generation of the correct header-code that has to be inserte into the page's <head> element. The purpose of this code is to contact the Google server to fetch all required, external JavaScript code used to render the charts. It also contains code, that will assign the data and options to the charts and to create the chart themselfs.

		getJavaScriptCode (line 105)
	
 
	
	

Genarates the JavaScript code that will prepare all charts.


This method will generate a preparation JavaScript-function for all charts  that have been added. This functions assign the chart's data, options and the  charts themselve to a global variable.  The generated code doesn't contain the required code to load the  packages from Google's server but a callback function that prepares and  displayes all charts automatically as soon as all packages have been loaded.

	
    
				
  • return: The generated JavaScript code.
    • 
				
  • throws: CodeGenerationException               Thrown, if the JavaScript code coudn't be generated, because               no data (Chart::setData()) has been assigned to at               least one chart.
