PieChart

Description
Description | Methods (details) Constants (details)

Creates a pie chart.

Data format:
The pie chart requires a DataTable which has two columns. The first column is used for the labels and and the second one for the values.

See https://google-developers.appspot.com/chart/interactive/docs/gallery/piechart for examples and detailed background information on the required data format.
