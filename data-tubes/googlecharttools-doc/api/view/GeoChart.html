GeoChart

Description
Description | Methods (details) Constants (details)

Creates a geo chart.

Data format:
The geo chart requires a DataTable which has two or three columns. The first column is used to set the region or the marker's position. Ihe second column is optional and can be used to set the region's or marker's color. In marker mode, the third column can be used to set the markers size.

See https://google-developers.appspot.com/chart/interactive/docs/gallery/geochart for examples and detailed background information on the required data format.
