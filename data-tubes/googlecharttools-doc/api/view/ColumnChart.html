ColumnChart

Description
Description | Methods (details) Constants

Creates a column chart.

Data format:
The column chart requires a DataTable with at least two columns. The first column is used for the x-axis labels and values. Each column that follows will be seen as y-values for one column.

See https://google-developers.appspot.com/chart/interactive/docs/gallery/columnchart for examples and detailed background information on the required data format.
